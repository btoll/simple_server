# simple_server

[![npm](https://img.shields.io/npm/v/simple_server.svg)](https://www.npmjs.com/package/simple_server)

## Installation

`npm i simple_server`

## Examples

Defaults to port 8080:

```
simple_server
```

Specify a port:

```
simple_server -p 65000
```

## Usage

Option | Description
------ | --------
-p | Port
